formangular

Data Driven Templates using Angular

Dynamic Templates are useful when you have view specific hints incorporated with content data.

For example some simple data... [ {"name": "1st simple demo", "url": "demo1.html", "tagline":"\"Keeping it simple\" will show the basic concept"}, {"title": "Some deatils about what DDT is and is not", "content": "What is happening ...", "subtext":"You may have missed some finer points..."}, {"name": "Diving in one level deeper", "url": "demo2.html", "tagline":"Start using hierarchical data"} ]

and some matching view specific hints... (in this case they are separate from the content) [ {"view_template": "demo_link"}, {"view_template": "demo_blurb"}, {"view_template": "demo_link"} ]

Finally ... the combined data and view hints [ {"view_template": "demo_link", "name": "1st simple demo", "url": "demo1.html", "tagline":"\"Keeping it simple\" will show the basic concept"}, {"view_template": "demo_blurb", "title": "Some deatils about what DDT is and is not", "content": "What is happening ...", "subtext":"You may have missed some finer points..."}, {"view_template": "demo_link", "name": "Diving in one level deeper", "url": "demo2.html", "tagline":"Start using hierarchical data"} ]
